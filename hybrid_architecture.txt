# Hybrid Chatbot Architecture

The Teethtalk Dental Clinic Pasig Chatbot utilizes a Hybrid Architecture that combines a custom Machine Learning intent classification model with Google's Gemini Generative AI.

## Flow:
1. **Input:** User provides a prompt.
2. **Intent Classification:** The prompt is vectorized (TF-IDF) and passed to the trained ML model (e.g., Logistic Regression, KNN, Naive Bayes).
3. **Probability Scoring (`predict_proba`):** The model calculates the confidence score (probability) for the predicted intent.
4. **Routing:**
   - **ML Fast-Path:** If the confidence score is >= 0.65 AND the predicted intent is NOT "general_inquiry" (i.e., it is a specific operational intent like "billing", "appointments", "post_op_care"), the system immediately returns a predefined, safe template response.
   - **Generative Fallback:** If the confidence score is < 0.65 OR the intent is "general_inquiry", the system forwards the prompt to Google Gemini (gemini-2.5-flash) to dynamically generate a helpful and contextual response.

This architecture ensures high accuracy and safety for critical clinic operations while maintaining conversational flexibility for general queries.
